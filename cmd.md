+ 切换目录：

    切换盘符可直接输入`e:`（*以切换到 E 盘为例*）。
    
    ```
C:\Users\HP>e:
    
    E:\>
    ```
    
    切换到该盘符下的目录。`cd`命令
    
    ```
    E:\>cd Microsoft VS Code
    
    E:\Microsoft VS Code>
    ```
    
    切换到其他盘符下的目录时在`cd`后加上`/d`
    
    ```
    c:\Users>cd /d e:\Microsoft VS Code
    
    e:\Microsoft VS Code>
    ```
    
    切换到上级目录。`cd..`
    
    
    
    注意，路径按下`tab`键有自动补全功能。
