## 数据表示方法

<span style="font-size:20px">定点数: </span>

n = 1(符号位) + n - 1 尾数

范围 [-2^n-1^, 2^n-1^ - 1]

> **最小整数为什么是 -2^n-1^ **而不是 -(2^n-1^ - 1)
>
> 按理说最小整数应该是 111...111
>
> 但 0 在二进制中有两种表示方法：
>
> +0 的原码：`0000 ... 0000`
>
> -0 的原码：`1000 ... 0000`
>
> 而 0 只需要一个，所以规定 -0 为最小的数，即 -2^n-1^



<span style="font-size:20px">科学计数法:</span>

$1e9+7=1000000007$ (8 个 0)
