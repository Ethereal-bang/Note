# 概述

## 操作系统历史

<span style="font-size:20px">第二代 晶体管和批处理系统:（多道）</span>

开启现代操作系统真正到来的时代

主要特征：多道性、并发性、无序性、调度性

缺点：无交互能力、作业平均周转周期长

<span style="font-size:20px">第三代 集成电路和多道程序:设计（分时系统）</span>

产生原因：人机交互能力需要、共享主机、终端技术成熟

特征：多路性、交互性、独占性、及时性

实现关键：及时接收、及时处理
